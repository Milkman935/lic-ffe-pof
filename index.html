How It Works
01
Generate Link
Select championship, year, and team above. Copy the generated link.
02
Send to Team
Email or message the link to the team. They open it and fill in quantities.
03
Team Submits
Team fills the form and hits Submit. Appears here as "Submitted".
04
Import to Matrix
Click "Import" to push team quantities into the main FFE matrix automatically.

⚠ Import Note: The "Import to Matrix" button works when both this admin page and the main matrix project (System/index.html) are served from the same local server, so they share the same localStorage.

Recommended: Run both from the same server root (e.g. python3 -m http.server 8080 from ~/Desktop), then open localhost:8080/POF/ and localhost:8080/System/. Alternatively, use the "Download JSON" button and upload via the matrix's Import button.
